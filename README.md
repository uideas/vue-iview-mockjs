#### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run dev
```

#### Compiles and hot reloads for test
```
npm run test
```

#### Compiles and minifies for production
```
npm run build
```

#### Lints and fixes files
```
npm run lint
```

#### Run with GUI
```
vue ui
```
> need install global vue-cli ```npm install -g @vue/cli```
